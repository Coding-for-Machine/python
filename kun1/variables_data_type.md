
## python dasturlash tilida ma'lumot turlari (Data types)

- [x] Matn turi(Text Type:) --> str <br>
- [x] Raqaml turi(Numeric Types:) --> int, float, complex<br>
- [x] Ko'rsatish turi(Mapping Type:) --> dict<br>
- [x] To'plam turi(Set Types:) --> set, frozenset<br>
- [x] Mantiq turi(Boolean Type:) --> bool
- [x] Binar ikkilik turi(Binary Types:) --> bytes, bytearray, memoryview
- [x] Yo'q turi(None Type:) --> NoneType

 ### ma'lumot turini aniqlash
 ```python
x = 5
y = "string"
z = ["olma", "anor"]
print(type(x))
print(type(y))
print(type(z))

```
